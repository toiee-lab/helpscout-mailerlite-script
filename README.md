# helpscout-mailerlite-script

HelpScout のサイドバーに、customer の email で検索した Mailerlite の情報を表示する

## 使い方

1. 適当な場所に設置
2. config.php.sample -> config.php
3. Mailelite の APIキーを取得、config.php に設定
4. HelpScout でカスタムApp登録、secret key を設定
5. config.php に設定
